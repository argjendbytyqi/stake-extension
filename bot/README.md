# StakePeek Blitz Telegram Bot

## Setup
1. Get a token from [@BotFather](https://t.me/BotFather).
2. Add it to `bot/.env`.
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the bot:
   ```bash
   python main.py
   ```

## Structure
- `main.py`: Entry point for the Telegram Bot.
- `.env`: Secret keys (Git ignored).
- `requirements.txt`: Python dependencies.
